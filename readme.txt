== Description ==

Frontend Manager makes easy and frontend better for you! It includes frontend dashboard, post edit, publishing, and WordPress user profile, post submissions.

<strong>FREE FEATURES</strong>

<strong>#1 Frontend Post Submission, edit and Profile Builder</strong>

Users has the ability to create new posts, edit post and edit their profile from the frontend, so that the user does not need to enter the backend admin panel to do action.

<strong>Update Profile from the Frontend</strong>

This Plugin allows registered users to edit their profile using default fields from frontend.

<strong>Featured Image & Image Upload</strong>

Users can upload images from the frontend. They can also upload featured image for a post using the Image Upload option on the post content area.

<strong>Use Forms Anywhere Easily with Shortcodes</strong>

All of the forms get an unique shortcode which you can paste on any page and the form will generate without breaking the style of your theme.

<strong>Custom Login & Registration Form</strong>

Registration Form Plugin also allows you to build registration forms using default templates.

== Installation ==

After having installed the plugin:
1. Click "Save" after installation for save settings.
1. Required pages will auto create while active the plugin. If not follow the following steps.
1. Create a new Page “Edit” for editing posts and insert shortcode `[wpfm-edit]`
1. Create a new page “Submit Post” for submit posts and insert shortcode `[wpfm-submit-post]`.
1. Create a new page “account” for account and insert shortcode `[wpfm-account]`.
1. To add Login feature, use the shortcode: `[wpuf-login]`
1. To enable a registration form in the frontend, use the shortcode: `[wpuf-register]`
1. Create a new Page “Dashboard” and insert shortcode `[wpfm-dashboard]`
1. Set the *Edit Page* option from *Others* tab on settings page.
